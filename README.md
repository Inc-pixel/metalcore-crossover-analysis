# Metalcore Crossover Analysis

Research question: When a metalcore song goes mainstream, what musical features change — and which ones predict it best?

## Dataset
30+ albums across 15 bands, manually annotated for breakdown density, vocal style, electronic production, guitar complexity, dropped tuning, and lyrical theme.

## Tools
Python, pandas, seaborn, scikit-learn
